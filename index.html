About

 Hi there, my name is Reid Luttrell. I have personal experience coding with several programming languages including C/C++, Python, Java. I have some experience with language processing models aswell -- I took a natural language processing course where our group fine-tuned several model checkpoints such as T5 on summarization using the HuggingFace library, and tested their performance in terms of Rouge and UniEval over several datasets. I also have personal experience with lower level memory architecture applications, and have worked with assembly to reverse engineer software. I've always been interested in video game interfacing and modification. For my purposes this involved modeling the games memory structure, and mapping custom functionality with a dynamically linked library and a new thread. This has always been an interesting process to me, because it involves a lot of topics such as memory architecture, machine instructions, and the results of your work tend to be very visual which is satisfying to me.
